## Visual Studio Code
---

### Tips

1. ALT + up-arrow/down-arrow
   * Move the code up/down
2. SHIFT + ALT + up-arrow/down-arrow
   * Duplicate the selected code
3. SHIFT + ALT + F
   * Format the code 
4. Ctrl + X
   * Cut line (empty selection)
5. Ctrl + C
   * Copy line (empty selection)
6. Ctrl + V
   * Paste
7. Ctrl + Z
   * Undo
8.  Ctrl + Y
    * Redo
9.  Ctrl + L
    * Select current line
